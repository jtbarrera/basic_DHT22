# basic_DHT22
Basic NodeMCU project using DHT22 to serial output. This will be used as basis to build additional feature sets. 

# Supported Features Include
- **DHT22** temperature sensor
- **DHT22** humidity sensor

# Parts List
- NodeMCU 1.0
- DHT22 Module
- breadboard
